## üî¨ Methodology

The analysis follows a rigorous standard for data science projects:

1.  **Data Source:** Uses an open-source, cleaned dataset for cardiovascular health parameters.
    * *Dataset Link: [Cardio Data](https://www.kaggle.com/aiaiaidavid/cardio-data-dv13032020)*
2.  **Preprocessing:** Includes necessary steps for handling missing values, encoding categorical features, and scaling numerical features.
3.  **Model Selection:** Utilizes **Random Forest** for its strong predictive power, ability to handle non-linear data, and built-in feature importance insights.
4.  **Hyperparameter Optimization:** Employs techniques (e.g., Grid Search or Random Search) to find the optimal set of parameters for the Random Forest algorithm.
5.  **Evaluation:** Focuses on key classification metrics like **Accuracy, Precision, Recall, and F1-Score** to assess model fitness for a high-stakes prognosis application.
